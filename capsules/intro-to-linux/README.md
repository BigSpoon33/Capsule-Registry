# Introduction to Linux

Learn the fundamentals of Linux — terminal navigation, file system, permissions, and basic commands. Perfect for absolute beginners.

## Prerequisites

None — this is a beginner course.

## What You'll Learn

- How to open and use a terminal emulator
- Basic Linux commands (ls, cd, pwd, mkdir, rm, cp, mv)
- The Linux file system hierarchy
- File permissions basics

## Course Structure

1. **Terminal Basics** — Getting comfortable with the command line
2. **The File System** — Understanding how Linux organizes files
3. **Lab: Navigation** — Hands-on practice navigating directories
4. **Quiz: Basics** — Test your knowledge
