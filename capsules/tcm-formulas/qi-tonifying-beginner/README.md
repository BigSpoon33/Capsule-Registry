# Qi Tonifying Formulas: Bensky Category

**Difficulty:** Beginner | **Estimated:** 3 hours | **Exams:** CALE, NCCAOM

Master the 4 foundational Qi-tonifying formulas from the Bensky category:

1. **Si Jun Zi Tang** (Four Gentlemen) — Foundational spleen qi tonic
2. **Shen Ling Bai Zhu San** — Spleen qi + internal dampness; chronic diarrhea
3. **Bu Zhong Yi Qi Tang** — Qi sinking; organ prolapse; lifting herbs
4. **Sheng Mai San** (Generate the Pulse) — Qi + yin deficiency; sweating + thirst
